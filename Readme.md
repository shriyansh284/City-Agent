🌆 City Intelligence AI Agent

An AI-powered assistant that provides real-time weather updates and latest news for any city using GenAI + LLM agents + API integration.

This project demonstrates how to build a tool-using AI agent that can interact with external APIs and deliver useful, real-world insights.

🚀 Demo

💬 Ask questions like:

“Weather in Delhi”
“Latest news in Mumbai”
“Weather and news of Jaipur”
✨ Features
🌦️ Real-time weather updates
📰 Latest news search
🤖 LLM Agent that decides which tool to use
💬 Interactive Streamlit chat UI
⚡ Clean and beginner-friendly implementation
🛠️ Tech Stack
LangChain – Agent & tool orchestration
Mistral AI – LLM reasoning
Streamlit – Frontend UI
OpenWeather API – Weather data
Tavily API – News search
🧠 Architecture
User Input → LLM Agent → Decide Tool → Call API → Return Response → UI
⚙️ Setup Instructions
1️⃣ Clone the Repository
git clone https://github.com/your-username/city-ai-agent.git
cd city-ai-agent
2️⃣ Install Dependencies
pip install -r requirements.txt
3️⃣ Setup Environment Variables

Create a .env file in root directory:

OPENWEATHER_API_KEY=your_openweather_key
TAVILY_API_KEY=your_tavily_key
4️⃣ Run the App
streamlit run Agent.py
📁 Project Structure
city-ai-agent/
│── Agent.py                # Main Streamlit app
│── .env                    # API keys
│── requirements.txt        # Dependencies
│── README.md               # Project documentation
📸 UI Preview

👉 Chat-based interface using Streamlit
👉 Simple and interactive
